NAME
====



`Log::Dispatch::Types` - standard types used by the `Log::Dispatch` module

DESCRIPTION
===========



Currently the only type this module provides is `LOG-LEVEL` enumeration (exported) with the following values:

  * `EMERGENCY`, which is *0*

  * `ALERT`

  * `CRITICAL`

  * `ERROR`

  * `WARNING`

  * `NOTICE`

  * `INFO`

  * `DEBUG`
